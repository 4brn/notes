## Overview

```
/ (root)
/bin (user binaries)
/sbin (system binaries)
/ets (configuration files)
/dev (device files)
/proc (process information)
/var (variable files)
/tmp (temporary files)
/usr (user programs)
/home (home directories)
/lib (system libraries)
/opt (optional add-on apps)
/mnt (mount directory)
/media (removable devices)
/srv (service data)
```
