# IP Address

IP addresses can be public (Internet) or private (in a Network).
IP addresses could be assigned manualy, physically or by the DHCP.

## IPv4

`192.168.0.1`

- The **IPv4** address consists of 4 octets (8 bytes each).
- Total number of _IPv4_ addresses = 2^32 = 4 billion (not enought for all devices)

## IPv6

`2a00:22c4:a531:c500:425f:c365:f64d`

- Total number of _IPv6_ addresses = 2^128 = 340 trillion
